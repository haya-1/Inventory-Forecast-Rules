# 供应商产能监控与占用操作手册

## 1. 适用范围

本文用于指导业务用户完成以下操作：

1. 维护供应商日均产能和产能空缺日。
2. 查看供应商产能日历和每日占用明细。
3. 在采购申请单中查看产能校验结果并调整需求交期。
4. 在成品采购单中完成产能校验、扣减、占用和释放。

## 2. 维护供应商产能

### 2.1 进入供应商产能信息

1. 进入“成品供应商”模块。
2. 打开需要维护的供应商档案。
3. 找到“产能信息”区域。

### 2.2 维护日均产能

1. 在“日均产能”字段中填写该供应商每天可承接的产能数量。
2. 确认数量无误后点击保存。
3. 保存后，系统会按最新日均产能重新计算当前日期之后的产能日历。

注意：

1. 当前日期之前的历史产能数据不会重新计算。
2. 如果保存后部分日期剩余产能变为负数，系统允许保存。
3. 剩余产能为负数的日期会在产能日历中以红色状态展示。

### 2.3 维护产能空缺日

1. 在供应商“产能信息”区域进入“产能空缺日”维护入口。
2. 新增供应商不可生产或不可分配产能的日期。
3. 保存产能空缺日。
4. 保存后，空缺日当天最大产能按 0 计算。

注意：

1. 系统会自动清理早于当前日期的产能空缺日。
2. 如果某个空缺日已经存在采购单占用，保存后该日期可能出现负数剩余产能，系统允许保存。
3. 后续采购单占用产能时，系统会跳过产能空缺日。

### 2.4 新增供应商后的产能日历

新增供应商保存后，系统会自动生成该供应商后续周期内的产能日历。业务用户无需手工初始化日历，只需要维护日均产能和产能空缺日。

## 3. 查看供应商产能

### 3.1 在供应商档案中查看

1. 进入“成品供应商”模块。
2. 打开供应商档案。
3. 在“产能信息”区域点击查看产能日历。
4. 在日历中查看每天的占用产能、剩余产能和状态。

### 3.2 在采购申请单中查看

1. 打开采购申请单。
2. 在分配供应商后，进入“调整需求交期”节点或相关处理页面。
3. 点击产能日历入口。
4. 查看所选供应商在目标交期附近的剩余产能。

### 3.3 在成品采购单中查看

1. 打开成品采购单创建页或详情页。
2. 选择供应商。
3. 点击供应商选项右侧的产能日历入口。
4. 查看该供应商产能日历。

注意：

1. 未选择供应商时，产能日历入口置灰，不可查看。
2. 发起确认、提交确认、确认货期等弹窗中也可以查看产能日历。

### 3.4 查看日期卡片

产能日历中的日期卡片主要展示：

1. 日期。
2. 占用产能。
3. 剩余产能。
4. 状态标签。

状态说明：

| 状态 | 含义 |
| --- | --- |
| 红色 | 剩余产能小于 0。 |
| 蓝色 | 当前日期。 |
| 灰色 | 剩余产能等于 0。 |

### 3.5 查看占用明细

1. 将鼠标移动到某个日期卡片上。
2. 系统展示当天产能占用明细。
3. 占用明细展示采购单号、SPU、占用数量。

## 4. 采购申请单中校验产能

### 4.1 创建采购申请单

1. 正常创建采购申请单。
2. 填写需求期望时间。
3. 创建时系统不限制需求期望时间。

### 4.2 分配供应商后校验

1. 完成供应商分配。
2. 系统根据供应商、采购量、需求期望时间和 SPU 生产周期自动校验产能。
3. 如果产能充足，系统自动跳过“调整需求交期”节点，进入“待下单”。
4. 如果产能不足，系统进入“调整需求交期”节点。

### 4.3 调整需求交期

1. 进入“调整需求交期”节点。
2. 打开产能日历，查看供应商剩余产能。
3. 将需求期望时间调整到产能充足的日期。
4. 提交调整。

提交结果：

1. 如果调整后的交期产能充足，可以提交。
2. 如果产能不足，系统不允许提交，并提示供应商剩余产能不足。

注意：

1. 采购申请单只做产能校验，不扣减、不锁定、不占用供应商产能。
2. 实际产能扣减发生在成品采购单确认流程中。

## 5. 成品采购单中占用产能

### 5.1 创建或查看采购单时查看产能

1. 进入成品采购单创建页或详情页。
2. 选择供应商。
3. 点击供应商右侧产能日历入口。
4. 根据预计交货时间查看供应商剩余产能。

### 5.2 发起确认时扣减产能

1. 在成品采购单中点击发起确认。
2. 系统根据供应商、采购总数、预计交货时间和 SPU 生产周期校验产能。
3. 如果产能不足，系统阻止发起确认，并提示：`该供应商剩余产能不足，请检查货期和数量后重试`。
4. 如果产能充足，系统发起确认并立即扣减供应商产能。

审批结果：

1. 审批通过后，本次扣减转为最终占用。
2. 审批拒绝后，系统退回本次扣减的产能。

### 5.3 提交确认时扣减产能

1. 在提交确认弹窗中确认即将提交的成品采购单。
2. 系统逐单校验供应商产能。
3. 产能充足的采购单正常提交并扣减产能。
4. 产能不足的采购单不提报，系统提示对应采购单产能不足。

审批结果：

1. 审批通过后，已扣减产能转为最终占用。
2. 审批拒绝后，系统退回已扣减产能。

### 5.4 确认货期时调整占用

1. 在成品采购单中执行确认货期。
2. 如果货期未变化，系统不重新计算产能。
3. 如果货期发生变化，系统重新校验产能。
4. 产能充足时，系统按新货期调整产能扣减。
5. 产能不足时，系统阻止确认，并提示：`该供应商剩余产能不足，请检查货期和数量后重试`。

审批结果：

1. 审批通过后，本次调整转为最终占用。
2. 审批拒绝后，系统回退本次产能调整。

### 5.5 修改货期或数量时重新占用

以下操作会触发产能重新校验：

1. 列表页执行货期变更。
2. 详情页编辑基础信息中的预计交货时间。
3. 详情页编辑货品详情。
4. 调整货品数量或采购总数。

操作方式：

1. 修改货期或数量。
2. 点击保存或提交。
3. 系统释放原占用产能，并按新货期、新数量重新校验和占用。
4. 如果新结果产能充足，操作成功。
5. 如果新结果产能不足，操作失败，并提示供应商剩余产能不足。

注意：

1. 如果交期和采购总数都没有变化，系统不重新计算产能。
2. 如果操作失败，原产能占用保持不变。

### 5.6 作废采购单时释放产能

1. 在成品采购单中执行作废操作。
2. 作废成功后，系统立即释放该采购单已占用的产能。
3. 释放后，对应日期的占用产能减少，剩余产能增加。

## 6. 产能占用规则说明

系统占用产能时，按以下方式处理：

1. 以预计交货时间作为目标交期。
2. 按 SPU 生产周期计算最早可占用日期。
3. 如果目标交期早于“当前日期 + 生产周期”，系统直接判定产能超限。
4. 如果目标交期可用，系统从目标交期当天开始倒序占用产能。
5. 目标交期当天产能不足时，继续向前一天占用。
6. 一直倒推到最早可占用日期。
7. 如果这段日期内仍然没有足够产能，系统判定产能超限。
8. 产能空缺日不会被用于新采购单占用。

## 7. 常见提示与处理方式

| 提示或现象 | 处理方式 |
| --- | --- |
| 未选择供应商，产能日历入口置灰 | 先选择供应商，再查看产能日历。 |
| `该供应商剩余产能不足，请检查货期和数量后重试` | 调整预计交货时间、需求期望时间或数量后重试。 |
| 采购申请单进入“调整需求交期”节点 | 打开产能日历，选择供应商产能充足的日期后提交。 |
| 产能日历某天显示红色 | 该日期剩余产能小于 0，可悬停查看占用明细。 |
| 批量提交时部分采购单不提报 | 根据提示中的采购单号，分别调整货期或数量后重试。 |
| 审批拒绝后产能恢复 | 系统已退回本次确认扣减，无需手动处理。 |
| 作废采购单后产能变化 | 系统已立即释放该采购单占用的产能。 |

## 8. 操作建议

1. 下采购申请单前，优先查看供应商产能日历，选择剩余产能充足的交期。
2. 供应商产能发生变化时，及时维护日均产能和产能空缺日。
3. 成品采购单发起确认前，确认预计交货时间和采购总数准确。
4. 需要修改已确认采购单的货期或数量时，先查看目标日期附近产能，避免保存失败。
5. 发现红色负产能日期时，优先查看悬停明细，确认是哪张采购单占用了产能。
