## 2. 背景与目标

### 2.1 现状

当前成品供应商支持按月维护产能，成品采购单会提示当月剩余产能。

### 2.2 问题

按月维护的供应商产能粒度较粗，无法有效指导运营在采购申请、供应商分配、采购下单、确认货期等场景中判断交期是否合理，也无法精确识别具体日期的产能占用与超限风险。

### 2.3 建设目标

1. 成品供应商支持维护日均产能和产能空缺日，并自动生成滚动的供应商产能日历。
2. 成品采购单在确认、提交、货期变更、数量变更、作废等场景中支持校验和占用供应商产能。
3. 采购申请单在分配供应商后增加“调整需求交期”节点，通过供应商产能日历辅助调整需求交期。
4. 支持在供应商、采购申请单、成品采购单相关页面查看产能日历和每日占用明细。
5. 基于 PLM 维护的 SPU 生产周期，从目标交期倒推可占用日期范围，避免交期早于合理生产周期。

## 3. 需求范围

### 3.1 范围内

1. 成品供应商产能信息模块改造：日均产能、产能空缺日、产能日历生成与展示。
2. 成品采购单产能占用：发起确认、提交确认、确认货期、货期变更、货品详情编辑、基础信息编辑、作废等场景。
3. 采购申请单流程改造：在“待分配供应商”和“待下单”之间新增“调整需求交期”节点。
4. 产能校验和占用规则：按供应商、日期、生产周期、需求数量进行日级产能判断。
5. 产能占用释放与重新占用：当交期、数量或采购单状态发生变化时更新占用结果。

### 3.2 相关依赖

1. PLM 需维护 SPU 级别的生产周期。
2. 产能日历需要记录每日最大产能、占用产能、剩余产能和占用来源。
3. 采购单占用明细需要支持反查，以便在产能日历中展示当天占用的采购单号、SPU、占用数量。

### 3.3 已确认补充口径

根据业务补充确认，本版整理稿采用以下口径：

1. 供应商档案保存时，若重新计算后出现剩余产能为负，允许保存。
2. 成品采购单发起确认时即实际扣减产能；审批拒绝则退回扣减，审批通过后转为最终占用。
3. 采购申请单仅做产能校验，不提前锁定或占用供应商产能。
4. PLM 生产周期按自然日计算。
5. 当目标交期早于“当前日期 + 生产周期”时，直接判定为产能超限。
6. 不存在多 SPU 的采购单。
7. 采购量、采购总数、货品数量与日产能为同一单位，不做单位换算。
8. 产能空缺日如已有采购单占用，设置后允许剩余产能为负数。
9. 采购单作废时立即释放产能。

## 4. 术语定义

| 术语 | 定义 |
| --- | --- |
| 日均产能 | 供应商在非产能空缺日的默认每日最大产能。 |
| 产能空缺日 | 供应商不可生产或不可分配产能的日期，该日期最大产能为 0。 |
| 最大产能 | 某供应商某日期可被分配的总产能。产能空缺日为 0，其他日期默认为日均产能。 |
| 占用产能 | 已被成品采购单等业务对象占用的产能数量。 |
| 剩余产能 | 最大产能减去占用产能。 |
| 产能日历 | 按供应商和日期展示最大产能、占用产能、剩余产能和占用明细的日历。 |
| 需求期望时间 | 采购申请单上的期望交付时间。 |
| 预计交货时间 | 成品采购单上的预计交付时间。 |
| 生产周期 | PLM 维护的 SPU 级别生产周期，用于计算可占用产能的最早日期。 |
| 产能超限 | 在允许日期范围内无法找到足够剩余产能，或目标交期早于“当前日期 + 生产周期”。 |

## 5. 总体业务规则

### 5.1 产能日历规则

1. 产能日历按供应商独立生成和分配。
2. 产能空缺日最大产能为 0。
3. 非产能空缺日每日最大产能等于供应商日均产能。
4. 剩余产能 = 最大产能 - 占用产能。
5. 产能占用时需要跳过产能空缺日。
6. 产能日历始终保持滚动 12 个月窗口。
7. 每次重新计算只计算当前日期之后、最长窗口期以内的日期。
8. 当前日期之前的历史产能数据不重新计算，最大产能也不随日均产能变动。

### 5.2 产能校验与占用算法

#### 5.2.1 输入

| 输入项 | 说明 |
| --- | --- |
| 供应商 | 被校验或占用产能的供应商。 |
| 需求数量 | 采购申请单采购量或成品采购单采购总数。 |
| 数量单位 | 采购量、采购总数、货品数量与日产能为同一单位，不做单位换算。 |
| 目标交期 | 采购申请单取需求期望时间，成品采购单取预计交货时间。 |
| 生产周期 | 根据 PLM 维护的 SPU 级别生产周期取得。 |
| 当前日期 | 系统当前日期。 |
| 产能日历 | 供应商在可用日期范围内的每日剩余产能。 |

#### 5.2.2 计算步骤

1. 计算最早可占用日期：当前日期 + 生产周期。
2. 若目标交期早于最早可占用日期，则直接判定为产能超限。
3. 计算可占用区间：最早可占用日期至目标交期，包含起止日期。
4. 从目标交期当天开始，按日期倒序占用剩余产能。
5. 若目标交期当天剩余产能不足，则继续向前占用前一日剩余产能。
6. 重复倒推，直到需求数量全部占用，或倒推至最早可占用日期仍不足。
7. 若可占用区间内产能不足，则判定为产能超限。
8. 若交期、采购总数均未变化，则不释放、不重新计算、不重新占用。

#### 5.2.3 示例

假设生产周期为 30 天，当前日期为 1 月 31 日，当前日期 + 生产周期为 3 月 3 日，采购申请单需求期望时间或成品采购单预计交货时间为 3 月 15 日。

系统只允许占用 3 月 3 日至 3 月 15 日之间的剩余产能，并优先占用 3 月 15 日剩余产能。若 3 月 15 日不足，则依次尝试 3 月 14 日、3 月 13 日、3 月 12 日，直到 3 月 3 日。若仍然不足，则视为产能超限。

### 5.3 产能释放与重新占用

以下场景需要释放原采购单已占用的产能，并按最新交期和数量重新占用：

1. 调整采购单交期。
2. 调整采购单货品数量或采购总数。
3. 确认采购单。
4. 已确认通过的采购单在详情页编辑基础信息中的货期。
5. 已确认通过的采购单在详情页编辑货品详情。
6. 列表页执行货期变更。

重新占用时若出现产能超限，则本次操作失败，并回滚至操作前状态。

作废采购单时立即释放该采购单已占用的产能。

## 6. 功能需求

### 6.1 成品供应商：产能信息模块

#### 6.1.1 日均产能维护

1. 成品供应商档案支持维护日均产能。
2. 日均产能用于生成供应商产能日历中非空缺日的每日最大产能。
3. 保存供应商产能信息时，需要触发当前日期之后窗口期内的产能日历重新计算。

#### 6.1.2 产能空缺日维护

1. 成品供应商档案支持维护产能空缺日。
2. 产能空缺日的每日最大产能为 0。
3. 产能占用时跳过产能空缺日。
4. 系统自动清理早于当前日期的产能空缺日。
5. 若将已有采购单占用的日期设置为产能空缺日，保存后允许该日期剩余产能为负数。

#### 6.1.3 产能日历自动生成

1. 每年 1 月 1 日自动生成接下来 12 个月的产能日历。
2. 每年 7 月 1 日自动生成接下来 12 个月的产能日历。
3. 新增供应商时，自动生成接下来的产能日历，结束时间与最近一次定时器触发生成的周期保持一致。
4. 新增供应商在 2 月或 6 月创建时，若其他供应商上一次生成到本年 12 月，则新增供应商也生成到本年 12 月。
5. 新增供应商在 7 月创建时，若其他供应商上一次生成到次年 6 月，则新增供应商也生成到次年 6 月。
6. 产能日历窗口始终保持 12 个月长度。

#### 6.1.4 产能日历重新计算

1. 保存供应商产能信息时触发重新计算。
2. 每次重新计算只计算当前日期之后、窗口期以内的数据。
3. 当前日期之前的数据保持不变，包括最大产能。
4. 非空缺日最大产能按最新日均产能计算。
5. 空缺日最大产能按 0 计算。
6. 占用产能根据采购单占用明细汇总。
7. 剩余产能根据最大产能减占用产能自动计算。

#### 6.1.5 产能日历展示

1. 产能日历以日期卡片形式展示。
2. 卡片主要展示日期、占用产能、剩余产能、状态标签。
3. 卡片不需要展示最大产能，顶部日均产能已体现默认产能水平。
4. 鼠标悬停在某日期时，展示当天占用产能的采购单号、SPU、占用数量。
5. 状态标签优先级如下：

| 优先级 | 条件 | 标签样式 |
| --- | --- | --- |
| 最高 | 剩余产能 < 0 | 红色 |
| 中 | 当前日期 | 蓝色 |
| 最低 | 剩余产能 = 0 | 灰色 |

#### 6.1.6 供应商档案负产能处理

供应商档案保存生成产能日历时，允许剩余产能为负数。

1. 若日均产能低于已占用产能日期的占用产能，保存后允许该日期剩余产能为负数。
2. 若产能空缺日已有占用产能，保存后允许该日期剩余产能为负数。
3. 剩余产能为负数的日期在产能日历中按红色状态标签展示。

### 6.2 成品采购单：产能校验与占用

#### 6.2.1 产能日历入口

1. 成品采购单创建页的供应商选项右侧支持查看产能日历。
2. 成品采购单详情页的供应商选项右侧支持查看产能日历。
3. 未选择供应商时，查看产能日历图标置灰。
4. 发起确认、确认货期等弹窗中支持查看产能日历。

#### 6.2.2 发起确认

1. 发起确认时，系统校验供应商产能日历。
2. 若产能超限，则阻止发起确认，并提示：`该供应商剩余产能不足，请检查货期和数量后重试`。
3. 若产能未超限，则正常发起确认，并立即实际扣减供应商剩余产能。
4. 若审批拒绝，则退回本次扣减的产能。
5. 若审批通过，则本次扣减转为最终占用。

#### 6.2.3 提交确认

1. 提交确认时，在提交弹窗中对即将提交的成品采购单逐单进行产能校验，并按发起确认规则立即扣减产能。
2. 若部分成品采购单产能不足，则这部分单据不提报。
3. 部分单据不足时，提示：`成品采购单AAA/BBB/CCC 对应供应商剩余产能不足，请检查预计到货时间和数量后重试`。
4. 若全部单据产能充足，则提交成功并进入审批；审批拒绝退回扣减，审批通过后转为最终占用。

#### 6.2.4 确认货期

1. 确认货期时，如果货期发生调整，则再次校验产能日历。
2. 若产能超限，则阻止确认，并提示：`该供应商剩余产能不足，请检查货期和数量后重试`。
3. 若产能未超限，则正常发起确认，并按新货期调整原本扣减的产能。
4. 若审批拒绝，则回退本次产能调整。
5. 若审批通过，则本次调整转为最终占用。
6. 提交确认时，如果部分单据出现产能不足，则不足单据不提报，并按提交确认规则提示。

#### 6.2.5 其他会改变交期或数量的操作

以下操作如果作用于已经确认通过的单据，也需要校验产能日历并重新占用：

1. 详情页基础信息中的货期编辑保存。
2. 详情页货品详情编辑保存。
3. 列表页货期变更按钮提交。
4. 调整货品数量。

若重新占用后产能超限，则操作失败，并提示：`该供应商剩余产能不足，请检查货期和数量后重试`。

作废采购单时立即释放该采购单已占用的产能。

#### 6.2.6 采购单批量提示口径

原始需求中存在两类批量提示：

1. `成品采购单AAA/BBB/CCC 对应供应商剩余产能不足，请检查预计到货时间和数量后重试`
2. `{采购单号1}、{采购单号2}产能超限，请调整交期或者数量后重试`

业务确认本期暂不统一批量超限提示文案，按各场景现有或原始文案处理；如后续需要统一，另行提出文案调整。

### 6.3 采购申请单：调整需求交期节点

#### 6.3.1 流程节点

1. 在“待分配供应商”和“待下单”之间新增“调整需求交期”节点。
2. 该节点为单级审批。
3. 审批人为固定人员，当前按原始需求记录为周燕英。

#### 6.3.2 创建采购申请单

1. 创建采购申请单时，仍然保留填写需求期望时间。
2. 创建时不对需求期望时间做产能限制。

#### 6.3.3 完成待分配供应商

1. 完成“待分配供应商”时，系统根据采购申请单的需求期望时间、供应商、采购量校验供应商产能日历。
2. 若校验通过，则自动跳过“调整需求交期”节点，进入“待下单”。
3. 若校验超限，则进入“调整需求交期”节点，由创建人调整为符合规则的交期。
4. 校验超限归入无法提报时，原因显示：`供应商剩余产能不足，请检查货期和数量后重试`。

#### 6.3.4 调整需求交期

1. 调整需求交期节点支持查看产能日历。
2. 提交调整后的交期时，需要结合供应商产能日历、数量、需求期望时间校验产能是否超限。
3. 校验超限时不允许提交。
4. 提交时只提交即将提报部分，不需要再次校验产能。
5. 采购申请单允许因并发导致提交后实际产能不足的情况。

#### 6.3.5 产能占用边界

采购申请单仅做产能校验，不提前锁定或占用供应商产能。实际产能扣减和最终占用由成品采购单流程完成。

### 6.4 PLM：SPU 生产周期

1. PLM 需新增或维护 SPU 级别生产周期。
2. 生产周期按自然日计算。
3. 产能校验时根据 SPU 生产周期计算最早可占用日期。
4. 当目标交期早于“当前日期 + 生产周期”时，直接判定为产能超限。
5. 不存在多 SPU 的采购单，因此不涉及多 SPU 生产周期取值规则。

## 9. 关键验收场景

1. 供应商维护日均产能后，系统可生成未来 12 个月产能日历。
2. 供应商维护产能空缺日后，空缺日最大产能为 0，产能占用时不分配到该日期。
3. 每年 1 月 1 日和 7 月 1 日可自动生成接下来 12 个月日历。
4. 新增供应商时，可按最近一次定时器周期生成对应结束日期的产能日历。
5. 成品采购单选择供应商后，可以从创建页和详情页查看产能日历；未选择供应商时入口置灰。
6. 成品采购单发起确认时，产能不足则阻止并提示。
7. 成品采购单审批拒绝时，已扣减的产能可回退。
8. 成品采购单确认货期发生变化时，系统释放原占用并按新交期重新占用。
9. 已确认通过的成品采购单修改预计交货时间或货品数量时，系统重新校验并占用。
10. 若交期和采购总数均未变化，则不触发释放和重新占用。
11. 采购申请单完成供应商分配后，产能充足时自动跳过“调整需求交期”节点。
12. 采购申请单完成供应商分配后，产能不足时进入“调整需求交期”节点。
13. 调整需求交期节点提交时，产能不足则不允许提交。
14. 产能日历卡片可展示日期、占用产能、剩余产能和状态标签。
15. 鼠标悬停在产能日历日期上，可查看当天占用产能的采购单号、SPU、占用数量。
16. 供应商档案保存后如出现剩余产能为负，允许保存，并在日历中展示负数状态。
17. 目标交期早于“当前日期 + 生产周期”时，系统直接判定为产能超限。
18. 采购申请单只做产能校验，不产生锁定或占用明细。
19. 采购单作废时立即释放已占用产能。

## 10. 补充确认记录

| 原问题 | 确认结果 |
| --- | --- |
| 供应商档案保存时，若重新计算后出现剩余产能为负，是否允许保存 | 允许保存。 |
| 成品采购单发起确认和提交确认的产能锁定边界 | 发起确认时实际扣减；审批拒绝退回扣减；审批通过后转为最终占用。 |
| 成品采购单批量超限提示文案是否统一 | 本期暂不统一。 |
| 采购申请单是否占用产能 | 仅做产能校验，不锁定、不占用。 |
| 采购申请单“调整需求交期”节点审批人 | 固定人员，当前按原始需求记录为周燕英。 |
| PLM 生产周期计算口径 | 按自然日计算。 |
| 目标交期早于“当前日期 + 生产周期”时如何处理 | 直接判定为产能超限。 |
| 多 SPU 采购单如何处理 | 不存在多 SPU 的采购单。 |
| 产能数量单位 | 采购量、采购总数、货品数量与日产能为同一单位。 |
| 已有按月产能数据迁移 | 本期不考虑。 |
| 产能并发处理策略 | 按成品采购单确认即扣减、审批拒绝退回、审批通过最终占用的策略处理。 |
| 产能空缺日已有采购单占用时如何处理 | 设置后允许剩余产能为负数。 |
| 原型链接或截图 | 本期不考虑。 |
| 产能日历悬停字段 | 展示采购单号、SPU、占用数量。 |
| 采购单作废释放时点 | 作废时立即释放产能。 |
