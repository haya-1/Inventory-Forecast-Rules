预警模块 · 钉钉机器人通知设计（ActionCard 版）

基于钉钉自定义机器人 ActionCard 消息类型的实际渲染能力设计，所见即所得。每张卡片下方附 JSON 报文，可直接用于开发对接。

ActionCard 能力与约束

消息类型：钉钉自定义机器人 → ActionCard（独立跳转按钮模式）

能做到的：
· 标题区：支持个性化文案，简短展示在钉钉消息列表中
· 内容区：支持 Markdown 子集 —— 加粗（**text**）、链接（[text](url)）、有序/无序列表
· 按钮区：支持 1~5 个独立跳转按钮，水平排列

做不到的（已在本设计中规避）：
· 不支持彩色文字 → 改用加粗标识异常值
· 不支持表格 → 改用「标签：值」逐行排列
· 不支持分割线 → 改用空行分段
· 不支持顶部色带

卡片预览 & JSON 报文

点击标签切换 7 种预警类型。左侧为钉钉内实际渲染效果，右侧为可直接使用的 ActionCard JSON 报文。
发送 / 跳转规则：
· 一父体一卡片：每张卡片只描述一个父体；一条规则命中 N 个父体就发 N 张独立卡片（不做聚合）。
· 库龄类卡片：同一父体内多个库龄阶段命中时，在同一张卡片里按阶段逐行展示。
· 不展示通知人、规则编号（仅用于后台投递与链接带参）。
· 查看详情：跳转到对应父体详情页（…/parent/detail?code={父体编号}）。
· 钉钉自定义机器人限流 20 条/分钟，父体数量多时由后台按队列限速发送。
